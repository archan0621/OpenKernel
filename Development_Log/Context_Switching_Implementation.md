# 컨텍스트 스위칭 구현 일지

> **작성일**: 2026-02-14  
> **구현 방식**: 인터럽트 기반 스케줄러 (ESP-only)

---

## 1. 뭘 만들었나?

여러 개의 태스크(프로그램)가 **번갈아가며 실행**되게 만들었어요.

**비유:**
- 선생님 1명이 학생 3명을 가르쳐요
- 철수 1분 → 영희 1분 → 민수 1분 → 다시 철수 1분...
- 각 학생은 "선생님이 나만 가르쳐주시는 것 같아!"라고 느껴요

**컴퓨터도 똑같아요:**
- CPU 1개가 프로그램 여러 개를 실행해요
- 아주 빠르게 바꿔가며 실행해서 동시에 실행되는 것처럼 보여요

---

## 2. 어떻게 만들었나?

### 방법: 타이머 알람 이용

```
1초마다 "딩동!" 알람이 울려요
    ↓
"다음 프로그램 차례!"
    ↓
프로그램을 바꿔요
    ↓
다시 알람이 울릴 때까지 실행
```

### 실제 코드로 보면

**1단계: 알람 울리면 현재 상태 저장**
```asm
pusha              ; 레지스터 8개를 스택에 저장
push ds, es, fs, gs ; 세그먼트도 저장
```
→ 마치 책에 **책갈피 끼우기**

**2단계: 다음 프로그램 찾기**
```c
scheduler_irq_handler() {
    old_task->esp = 현재_스택;  // 지금 위치 기억
    new_task = 줄_서있는_첫번째;  // 다음 차례 찾기
    return new_task->esp;        // 새 위치 알려주기
}
```
→ 마치 **다음 학생 부르기**

**3단계: 새 프로그램으로 바꾸기**
```asm
mov esp, eax       ; 스택을 바꿔요!
```
→ 마치 **책상 바꾸기**

**4단계: 새 프로그램 상태 복원**
```asm
pop gs, fs, es, ds ; 세그먼트 복원
popa               ; 레지스터 복원
iret               ; 실행 재개!
```
→ 마치 **새 책의 책갈피부터 다시 읽기**

## 3. 핵심 아이디어: "ESP만 바꾸면 끝!"

### ESP가 뭔가요?
**ESP = 스택 포인터 = "지금 어디 보고 있는지" 알려주는 손가락**

```
책상 위에 공책이 쌓여있어요:

[공책5] ← ESP가 여기를 가리켜요!
[공책4]
[공책3]
[공책2]
[공책1]
[책상]

ESP를 바꾸면? → 다른 공책 더미를 가리키게 돼요!
```

### 왜 ESP만 바꾸면 될까?

모든 정보가 **스택에 저장**되어 있기 때문이에요!

```
Task A의 스택:
[레지스터들...]
[프로그램 위치...]
[기타 정보...]

Task B의 스택:
[레지스터들...]
[프로그램 위치...]
[기타 정보...]

ESP를 바꾸면?
→ 스택 전체가 바뀌어요!
→ 모든 정보가 바뀌어요!
```

## 4. 주의해야 할 점들

### 주의 1: 스택은 "접시 쌓기"

```
접시를 쌓으면:
1번 접시 (바닥)
2번 접시
3번 접시 (맨 위)

꺼낼 때는:
3번 접시 (먼저 꺼냄!)
2번 접시
1번 접시 (나중에 꺼냄!)
```

**코드에서:**
```asm
push ds    ; ds를 먼저 넣음
push es    ; es를 그 위에 넣음
push fs
push gs    ; gs가 맨 위!

pop gs     ; gs를 먼저 꺼냄!
pop fs
pop es
pop ds     ; ds를 나중에 꺼냄!
```

### 주의 2: PUSHA 순서

**Intel CPU는 이 순서로 저장해요:**
```
EAX 먼저 → ECX → EDX → EBX → ESP → EBP → ESI → EDI 마지막

스택에서는:
[EDI][ESI][EBP][ESP][EBX][EDX][ECX][EAX]
 ↑맨위                              ↑맨아래
```

**새 태스크 만들 때도 같은 순서로 넣어야 해요!**

### 주의 3: ESP가 가리키는 곳

```c
// ❌ 틀린 방법
task->esp = stack_top;  // 빈 공간을 가리킴

// ✅ 맞는 방법  
*(--sp) = 값1;
*(--sp) = 값2;
...
task->esp = sp;  // 데이터가 있는 곳을 가리킴!
```
